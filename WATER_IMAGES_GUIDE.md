### Tips for Selecting Images
1. **Relevance**: Ensure the image matches the section theme
2. **Quality**: Use high-resolution images (at least 1920px wide for backgrounds)
3. **Consistency**: Maintain a consistent style across the website
4. **Cultural Sensitivity**: Use images that respectfully represent African communities
5. **Format**: Save as JPG (for photos) or PNG (for graphics)
6. **File Size**: Optimize images to 200-400KB for web performance

## How to Replace Images

### Step 1: Download New Images
1. Visit one of the free stock photo sites above
2. Search for the recommended terms
3. Download the image in high resolution

### Step 2: Resize/Optimize (Optional)
Use tools like:
- **TinyPNG** (tinypng.com) - Free image compression
- **ImageResizer** (imageresizer.com) - Resize images
- **Photoshop** or **GIMP** (free alternative) - Professional editing

### Step 3: Replace Files
1. Go to the `/home/dalton/Projects/Gaia Africa new/images/` folder
2. Backup the original image (rename with "_old" suffix)
3. Replace with your new image, keeping the same filename

### Step 4: Test
1. Reload the website in your browser
2. Check all pages to ensure images display correctly
3. Verify responsive design on mobile devices

## Example: Replacing bg_1.jpg
